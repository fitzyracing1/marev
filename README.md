# Marev1 (Rust + Base)

MAREV is a Base crypto platform with a live site, token discovery DEX, meme coin factory, Fire Coin trading flow, ATTN rewards, and public builder tools.

## Live Links

- Home: `https://marev-bay.vercel.app/`
- DEX: `https://marev-bay.vercel.app/dex.html`
- Meme Coin Factory: `https://marev-bay.vercel.app/factory.html`
- Attention Claims: `https://marev-bay.vercel.app/attention.html`
- AI Builders Page: `https://marev-bay.vercel.app/ai.html`
- Web CLI: `https://marev-bay.vercel.app/cli.html`
- Public Editor: `https://marev-bay.vercel.app/editor.html`

## Rust Demo

Minimal Rust cryptocurrency/blockchain demo with proof-of-work and transactions.

## Quick Start

```powershell
cargo run
```

## Tests

```powershell
cargo test
```

## Structure

- `src/block.rs` - Block structure + mining
- `src/blockchain.rs` - Chain management
- `src/transaction.rs` - Transactions + fee
- `src/wallet.rs` - Wallet addresses
- `src/main.rs` - Demo runner
