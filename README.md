# Marev1 (Rust)

Minimal Rust cryptocurrency/blockchain demo with proof-of-work and transactions.

## Quick Start

```powershell
cargo run
```

## Tests

```powershell
cargo test
```

## Structure

- `src/block.rs` - Block structure + mining
- `src/blockchain.rs` - Chain management
- `src/transaction.rs` - Transactions + fee
- `src/wallet.rs` - Wallet addresses
- `src/main.rs` - Demo runner
