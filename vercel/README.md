# Marev1 Mainnet (Vercel)

Static Vercel site that connects MetaMask and verifies Ethereum Mainnet.

## Deploy to Vercel

- Set the project root to `vercel/`.
- Framework preset: **Other** (static).
- Build command: **none**
- Output directory: **/**

## Local Preview

Open `index.html` in a browser with MetaMask installed.

## Features

- Connect MetaMask
- Switch to Ethereum Mainnet
- Display ETH balance
