// AUCTION PARAMETERS

For a Dutch auction, Start Price is the most expensive (where the auction begins) and End Price is the cheapest (where it ends). Buyers walk the price down through the range as they swap. The auction "activates" only when the spot pool price reaches your start price — for option C we set the range right at the current pool price, so it's live from minute zero.
